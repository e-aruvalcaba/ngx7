CEMEX Digital Design Framework

All the brand guidelines, graphics and components of CEMEX Digital World, under one umbrella. This is a framework of best practices and a living artifact of art direction, documentation and code snippets maintained by CEMEX.

Our mission is to build consistent experience, translated into apps, that conform to our users and market needs. Under each category you'll find the latest examples and live demos of how to apply the styles in order to achive this scope.
