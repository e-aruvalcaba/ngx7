CEMEX Digital Design Framework - RTL

In a right to left layout, some components are different. Below are links to illustrate by example the look of these elements.

From a code point of view, the change is made at the top level of the DOM (Document Object Model), by specifing on the html tag, the attribute "dir" (from direction) with a value of "rtl" (right to left).
