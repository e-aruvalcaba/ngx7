Numeric Filter

The Numeric Filter is designed to select a range (Min - Max) of numeric data.

There are two options depending on the Screnario:

  • Numeric Filer with Slider:
    Use it when the user doesnÂ´t need an exact min or max number, only and aproximation
  • Numeric Filter with Text Input:
    Use it when the user needs to input a specific range, i.e. Min = 10.23, Max = 354.21

Note: for the implementation of the double range selector the plugin nuoSlider was used. You can check out the specs of it here.

<script>
    // pick your slider element
    var zeSlider = document.querySelector('#zeSlider');

    // define the props of it
    var sliderProps = {
        start: [ 500,5000 ],
        connect: true,
        range: {
            'min': 0,
            'max': 7000
        },
        labelPre:'$',
        labelPost:''
    }

    // you shouldn't be editing past here, but feel free to do it if you want.

    noUiSlider.create(zeSlider, sliderProps);

    var rangeTagLower = document.createElement('div');
    var rangeTagHigher = document.createElement('div');
    rangeTagLower.classList.add('cmx-num-filter__range-label');
    rangeTagHigher.classList.add('cmx-num-filter__range-label');
    zeSlider.querySelector('.noUi-handle-lower').appendChild(rangeTagLower);
    zeSlider.querySelector('.noUi-handle-upper').appendChild(rangeTagHigher);

    zeSlider.noUiSlider.on('update', function( values, handle ) {
        var value = values[handle];
        if ( handle ) {
            rangeTagHigher.innerHTML = sliderProps.labelPre + Math.round(value) + sliderProps.labelPost;
        } else {
            rangeTagLower.innerHTML = sliderProps.labelPre + Math.round(value) + sliderProps.labelPost;
        }
    });
</script>
