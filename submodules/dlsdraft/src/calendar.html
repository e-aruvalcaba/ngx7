Calendar

Date Selection

Shown are multiple states for the calendar date range selection.

The calednar date selection should be used in tables to select a date range to be viewed.

The user can select a single day by selecting a single day and then selecting "apply".

If they would like to selet a date range they can simlpy start to hover and move their mouse to make a second selection and then once then selection is made, click apply.
