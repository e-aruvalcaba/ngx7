Lightbox

Note: The implementation of the lightbox uses the same exact code as the Modal Component. Below are more details on the apperance and the behaviour of this specific element.

Details Lightbox with Table

After the user clicks on the icon, there should always display a More Details Lightbox, this provides transparency to the user throughout the experience interacting with the web app.

General Guidelines

  • More details lightbox should contain: Call to action buttons and a close icon on the upper right hand corner.
  • The top portion of the lightbox should display the main information such as Location , Creation Date, Location ID, and Product Line.
  • Each modal will have to follow this format and have Call to Action buttons at the botttom of it.
